【执行规则】
1. 本节点只能做同类项目经验借鉴，不能替代本项目实际工程分析、污染源强核算、排放标准适用和治理措施设计。
2. 同类项目资料必须来自 {evidence_context}、上传资料或实际联网检索得到的可核验 URL；不得编造同类项目名称、报告内容、产污节点、治理效率或达标结论。
3. 如未检索到足够相似的项目报告，应说明相似性不足，输出“资料不足，建议人工核实”，并给出后续资料收集建议。
4. 对同类项目与本项目的行业、产品、产能、工艺、原辅材料、污染物、治理设施差异必须单独说明；差异较大时不得直接套用治理措施。
5. 引用同类项目污染节点和治理措施时，应标明来源文件或 URL、章节或可见摘要；不要粘贴大段报告原文。

【输出结构】
一、同类项目资料来源与可比性
请按“资料名称 / 来源 / 项目类型 / 与本项目相似点 / 差异点 / 可借鉴程度 / 有效性或可信度”输出。

二、同类项目主要工艺流程及产污节点
请梳理主要工艺环节、产污节点和对应污染因子，并说明与本项目的对应关系。

三、污染源与治理措施借鉴
请分别输出废气、废水、噪声、一般固废、危险废物、土壤地下水、环境风险等污染源、常见治理措施、达标可行性和适用限制。

四、与本项目的相似点和差异点
请重点比较产品方案、产能规模、生产工艺、原辅材料、污染物类型、治理设施、排放去向和环境风险源。

五、对本项目资料收集和工程分析的建议
请列出需要补充的工艺流程图、物料平衡、水平衡、VOCs 物料、废水去向、危废属性、治理设施设计参数、排放标准和监测数据。

六、风险点、需补充资料、人工复核要点
请按“风险事项 / 风险等级 / 依据 / 影响 / 建议”输出，并明确“同类项目经验仅供参考，不能替代本项目实际工程分析”。

【固定免责声明】
输出末尾必须追加：以上为AI辅助初步研判，最终结论需由环评工程师人工复核确认。

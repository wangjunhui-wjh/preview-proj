【用户提示词 - HB-PT-004 规划及规划环评符合性分析】

请调用 HB-PT-001 项目概况提取结果，并结合知识库或联网检索获取的所在区域规划、园区规划、规划环评及审查意见文件，进行规划符合性分析。

【输入变量】
项目概况：
{project_profile}

知识库依据片段：
{evidence_context}

【执行规则】
1. 项目选址、用地性质、园区名称、产业类型、建设规模、工艺和污染物排放特征必须来自 {project_profile} 或已上传项目资料。
2. 规划、规划环评、审查意见、准入清单、负面清单、总量控制和基础设施要求必须来自 {evidence_context}、实际读取的上传资料或实际联网检索得到的官方文件。
3. 如缺少所在区域规划、园区规划、规划环评报告或审查意见，不得凭经验判断符合；应输出“资料不足，建议人工核实”，并列明需要补充的文件。
4. 如联网检索未取得规划原文，只能记录可核验的官方 URL、标题和摘要，不得编造规划文本、审查意见文号或条款。
5. 对用地性质、产业定位、准入清单、负面清单、总量控制、污水处理/集中供热/危废处置等基础设施条件分别给出单项结论；依据不足的单项写“资料不足，建议人工核实”。

【输出结构】
一、规划符合性总判断
请在“符合 / 基本符合但需补充核实 / 存在不符合风险 / 资料不足”中选择，并说明主要依据。

二、总体规划符合性
请分析项目选址、用地性质、产业定位、建设规模与所在区域总体规划、园区规划或控制性详细规划的匹配情况。

三、规划环评及审查意见符合性
请按“规划环评要求 / 项目对应事实 / 符合性判断 / 依据 / 风险等级”输出，重点包括准入清单、负面清单、总量控制、污染物排放、环境风险和基础设施依托条件。

四、基础设施建设条件符合性
请分析污水处理厂、供热或供气、危废暂存与处置、事故应急、集中治理设施等条件是否支撑项目建设。

五、所引用规划文件有效性识别
请按“文件名称 / 文号或审查意见 / 发布或审查时间 / 来源 / 效力状态 / 是否建议用于本次研判”输出；无法确认效力的写“有效性待核实，建议人工确认”。

六、风险点、需补充资料、人工复核要点
请按“风险事项 / 风险等级 / 依据 / 影响 / 建议”输出，并列明需要补充的规划文件、规划环评报告、审查意见、园区准入或用地证明材料。

【固定免责声明】
输出末尾必须追加：以上为AI辅助初步研判，最终结论需由环评工程师人工复核确认。
